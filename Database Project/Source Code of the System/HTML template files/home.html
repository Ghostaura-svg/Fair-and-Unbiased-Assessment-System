A Complete Academic Assessment Workflow

The system connects Admin, Faculty, and Student modules with a normalized MySQL database. Each record is stored with proper relationships so the assessment process remains organized and traceable.

1
Admin manages academic setup

Students, faculty, courses, sections, teacher assignments, and enrollments are managed from the admin panel.

2
Faculty controls assessment portals

Faculty can create assessments, open or close portals, review submissions, and manage evaluations.

3
Students submit and view results

Students can upload submissions while portals are open and view reports after evaluation.
